Teoria dos Conjuntos

Conjunto unitário - Conjunto vazio

Um conjunto unitário tem um único elemento.

Exemplos:

  1.   conjunto dos divisores de 1, inteiros e positivos: {1}
  2.   conjunto dos estados brasileiros que começam com a letra B: {Bahia}

Um conjunto vazio não possui elementos. O conjunto vazio é representado por Ø.

Exemplos:

  1.   {x | x ≠ x} = Ø
  2.   {x | x > 1 e x < 1} = Ø

Conjunto universo

O conjunto universo é representado pelo conjunto U e é composto de todos os elementos de um determinado assunto da Matemática. Portanto, o conjunto universo possui todos os valores que a variável pode assumir.

Por exemplo, para os números inteiros positivos temos U = {1, 2, 3, ...}.

Subconjuntos

Um conjunto B é subconjunto de um conjunto C se, e somente se, todo elemento de B também pertence a C.

Dizemos então que "B é subconjunto de C" ou "B está contido em C" com a seguinte notação: B ⊂ C.

Logo,

B ⊂ C ⇔ (∀ x) (x ∈ B ⇒ x ∈ C)

Exemplos:

  1.   {1, 2} ⊂ {1, 2, 3, 4}
  2.   {5} ⊂ {3, 5, 7}
  3.   {b, c} ⊂ {b, c}

Pode-se também escrever C ⊃ B, que significa: "C contém B".

Com a notação B ⊄ C indicamos a negação de B ⊂ C, ou seja, dizemos que "B não está contido em C".

B ⊄ C é verdade quando existe pelo menos um elemento de B que não pertence a C.

Exemplos:

  1.   {1, 2, 3} ⊄ {2, 3, 4, 5}
  2.   {b, c} ⊄ {d, e, f}

Se B ⊂ C e C ⊂ B, então B = C.

Propriedades da inclusão

Sendo B, C e D três conjuntos, valem as seguintes propriedades:

  1. ∅ ⊂ B
  2. B ⊂ B (reflexiva)
  3. (B ⊂ C e C ⊂ B) ⇒ B = C (antissimétrica)
  4. (B ⊂ C e C ⊂ D) ⇒ B ⊂ D (transitiva)

Conjuntos iguais

Dois conjuntos são iguais quando todos os elementos de um dos conjuntos são encontrados no outro conjunto, e vice-versa, independentemente da ordem. Ou seja,

A = B ⇔ (∀ x) (x ∈ A ⇔ x ∈ B}

Exemplos:

  1.   {1, 2, 3, 4} = {4, 3, 2, 1}
  2.   {2, 4, 6, 8, ...} = {x | x é inteiro, positivos e par}
  3.   {x | 3x - 1 = 8} = {3}

Não há necessidade da repetição de um elemento na descrição de um conjunto. Isto significa que:

{a, b, c, d} = {a, a, b, b, c, d, d, d}

Os conjuntos não são iguais quando existe um elemento em A que não pertence a B ou existe um elemento em B que não pertence a A.

Exemplo:

{a, b, c} ≠ {a, b, c, d, e}

Conjunto das partes

Chama-se conjunto das partes de um conjunto B - notação P(B) - aquele que é formado por todos os subconjuntos de B. Em símbolos:

P(B) = {x | x ⊂ B}

Exemplos:

  1.   Se B = {c}, os elementos de P(B) são ∅ e {c}, ou seja:   P(B) = {∅, {c}}.
  2.   Se B = {c, d}, os elementos de P(B) são ∅ e {c}, {d} e {c, d}, ou seja:   P(B) = {∅, {c}, {d}, {c, d}}.
