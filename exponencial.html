Função Exponencial

Sendo a um número real, tal que 0 < a ≠ 1, denominamos função exponencial a função f de R em R de base a que associa a cada x real o número ax.

Exemplos:

1) f(x) = 3x

2) g(x) = 5x


Propriedades

1°) Se x = 0, temos na função exponencial f(x) = ax:

f(0) = a0 = 1

ou seja, o par ordenado (0, 1) pertence à função para todo a ∈ R+ - {0, 1}.

2°) Se a > 1 a função exponencial f(x) = ax é crescente, e se 0 < a < 1 a função é decrescente. Logo, sendo x1 e x2 reais, temos:

  1. quando a > 1:           x1 < x2 ⇒ f(x1) < f(x2)
  2. quando 0 < a < 1:     x1 < x2 ⇒ f(x1) > f(x2)

3°) A função f(x) = ax, com 0 < a ≠ 1, é injetora, pois, com x1 e x2 tais que x1 ≠ x2, temos:

  1. se a > 1: f(x1) < f(x2);
  2. se 0 < a < 1: f(x1) < f(x2);

e, logo, f(x1) ≠ f(x2).

Equações exponenciais


São equações com a incógnita no expoente.

Podemos utilizar o método da redução a uma base comum para resolver uma equação exponencial.
Para isso, basta reduzir os membros da equação a potências de mesma base a (0 < a ≠ 1).

ax = ay ⇔ x = y (0 < a ≠ 1)

Exemplos:

  1. 2x = 32 ⇔ 2x = 25 ⇔ x = 5
  2. 9x = 27 ⇔ (32)x = 33 ⇔ 2x = 3 ⇔ x = 3/2

Inequações Exponenciais


São as inequações que possuem incógnita no expoente.

Continuaremos a utilizar o método de redução para potências de mesma base a (0 < a ≠ 1).

Logo:   Se x e y são números reais, então:
           para x > 1 tem-se ax > ay ⇔ x > y
           para 0 < a < 1 tem-se ax > ay ⇔ x < y

Exemplos:

1. 2x > 64   ⇔   2x > 26
    Como a base é maior que 1, x > 6.
    S = {x ∈ R | x > 6}


    Como a base está compreendida entre 0 e 1, x ≤ -3.
    S = (x ∈ R | x ≤ -3}
