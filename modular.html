Definição de módulo

Chama-se módulo ou valor absoluto de x, indicado por |x|, por meio da seguinte relação:

|x| = x   se x ≥ 0
ou
|x| = -x   se x < 0

Assim:

  • o módulo de um número real não negativo é igual ao próprio número;
  • o módulo de um número real negativo é igual ao oposto dele.

Propridades

  1. |x| ≥ 0, ∀ x ∈ R
  2. |x| = 0 ⇔ x = 0
  3. |x| . |y| = |xy|, ∀ x, y ∈ R
  4. |x|² = x², ∀ x ∈ R
  5. x ≤ |x|, ∀ x ∈ R
  6. |x + y| ≤ |x| + |y|, ∀ x, y ∈ R
  7. |x - y| ≥ |x| - |y|, ∀ x, y ∈ R
  8. |x| ≤ a   e   a > 0 ⇔ -a ≤ x ≤ a
  9. |x| ≥ a   e   a > 0 ⇔ x ≤ -a ou x ≥ a

Função Composta

Sendo f uma função de um conjunto A em um conjunto B e sendo g uma função de B em um conjunto C. Função composta de g e f é a função h de A em C em que se obtém a imagem de cada x da seguinte forma:

  1. A função f é aplicada a x, obtendo-se f(x);
  2. A função g é aplicada a f(x), obtendo-se g(f(x)).

Assim, h(x) = g(f(x)) para todo x ∈ A

A composta pode ser indicada por g ο f, logo: (g ο f)(x) = g(f(x)) para todo x ∈ A.

A composta g ο f pode ser representada pelo seguinte diagrama:

Exemplo:

Dados os conjuntos A = {-2, -1, 0, 1}, B = {1, 2, 3,} e C = {2, 4, 6, 8} e a seguintes funções:

  • f, de A em B, definida por f(x) = 2x;
  • g, de B em C, definida por g(x) = 3x - 1.

Para h(1) obtemos 5, pois h(1) = (g ο f)(1) = g(f(1)) = g(2) = 5.

Conseguimos a lei de correspondência da função composta h = g ο f, trocando x por f(x) na função g(x). Assim obtemos g(f(x)).

Para nosso exemplo, temos:

h(x) = (g ο f)(x) = g(f(x)) = 3 . f(x) - 1 = 6x - 1

Fazemos assim para calcular h(1):

h(1) = 3 . 2(1) - 1 = 5

OBS.: (h ο g) ο f = h ο (g ο f)

Função sobrejetora

Uma função f de A em B é sobrejetora se, e somente se, pelo menos um elemento x pertencente a A estiver ligado com um elemento y pertencente a B.

O contradomínio é sempre igual ao conjunto imagem.

Logo, f: A → B é sobrejetora se, e somente se, Im(f) = B.

Exemplo:

Uma função f de A = {0, -2, 2, 3} em B = {0, 4, 9} definida pela lei f(x) = x2 é sobrejetora, com base nas definições acima.

Função injetora

Uma função f de A em B é injetora se, e somente se, cada elemento x pertencente a A possuir apenas um correspondente y pertencente a B.

Exemplo:

Uma função f de A = {2, 3, 4} em B = {3, 6, 9, 12} definida pela lei f(x) = 3x - 3 é injetora, com base na definição acima.

Função bijetora

Uma função f de A em B é bijetora quando ela for sobrejetora e injetora.

Exemplo:

Uma função f de A = {2, 3, 4} em B = {4, 5, 6} definida pela lei f(x) = x + 2 é bijeora, com base na definição acima.

Função inversa

Definição

Dada uma função f bijetora de A em B, a relação inversa é uma função de B em A chamada de função inversa de f e é indicada por f-1.

(x, y) ∈ f ⇔ (y, x) ∈ f-1

O domínio da função f-1 é B, que é a imagem da função f.

A imagem da função f-1 é A, que é o domínio da função f.

Exemplo:
