Limite

Noção de limite

Analisemos os valores da função quando x assume valores próximos de 2, mas diferentes de 2.

Com valores próximos de 2, porém menores que 2, temos:

x 0 1,5 1,75 1,9 1,99 1,999
f(x) 1 5,5 6,25 6,7 6,97 6,997

E com valores próximos de 2, porém maiores que 2, temos:

x 3 2,5 2,25 2,1 2,01 2,001
f(x) 10 8,5 7,75 7,3 7,03 7,003

Quando x se aproxima de 2, f(x) aproxima-se cada vez mais de 7, com isso, quanto mais próximo x estiver de 2, tanto mais próximo f(x) estará de 7.

Podemos tornar f(x) mais próximo de 7 o quanto quisermos, basta tomarmos x suficientemente próximo de 2.

Em linguagem matemática utilizamos símbolos para indicar pequenas essas pequenas diferenças. Os símbolos utilizados são ε (épsilon) e δ (delta).

Assim, se quisermos que |f(x) - 7| seja menor que ε, devemos tomar |x - 1| suficientemente pequeno, ou seja, encontrar um número positivo δ, suficientemente pequeno, de tal modo que

0 < |x - 2| < δ  ⇒  |f(x) - 7| < ε

Perceba que δ depende do ε considerado. Tomando as tabelas vemos que:

1°) x = 2,1   ⇒   f(x) = 7,3   assim, x - 2 = 0,1   ⇒   f(x) - 7 = 0,3
     |x - 2| = 0,1   ⇒   |f(x) - 3| = 0,3
     então, se ε = 0,3, δ = 0,1 e afirmamos que:

0 < |x - 1| < 0,1   ⇒   |f(x) - 7| < 0,3

1°) x = 2,01   ⇒   f(x) = 7,03   assim, x - 2 = 0,01   ⇒   f(x) - 7 = 0,03
     |x - 2| = 0,01   ⇒   |f(x) - 3| = 0,03
     então, se ε = 0,03, δ = 0,01 e afirmamos que:

0 < |x - 1| < 0,01   ⇒   |f(x) - 7| < 0,03

Definição de limite

Seja I um intervalo aberto ao qual pertence o número real a. Seja f uma função definida para x ∈ I - {a}.

Não é necessário que a função esteja definida em a, por isso que nada é mencionado sobre o valor da função quando x = a.

Lembre-se sempre que no cálculo de limite o que interessa é o comportamento de f(x) quando x se aproxima de a e não o que acontece com f quando x = a.

Unicidade do limite

Se o limite de uma função existe, então ele é único.

Teorema

Limites laterais

Antes estávamos interessados no comportamento da função quando os valores estavam próximos de a, ou seja, quando x assumia um valor pertencente a um intervalo aberto contendo a mas não sendo igual a a, e sim, nos valores do intervalo que são maiores ou menores que a.

Todavia, em algumas funções, o comportamento quando x está próximo de a mas com valores menores que a, é diferente do comportamento quando x assume valores maiores e próximos de a.

Exemplo:

para valores menores que 1, porém próximos a 1, temos:

x 0 0,5 0,75 0,9 0,99 0,999
f(x) 6 5,5 5,25 5,1 5,01 5,001

e para valores maiores que 1, porém próximos a 1, temos:

x 2 1,5 1,25 1,1 1,01 1,001
f(x) -1 -1,5 -1,75 -1,9 -1,99 -1,999

Podemos observar que os valores da função estão próximos de 5 quando x está próximo e é menor que 1, e os valores da função estão próximos de -2 quando x está próximo e é maior que 1.

Consideramos os limites laterais quando supomos x assumindo valores próximos de 1, porém menores ou maiores que 1.

Definição

Se f é uma função definida em um intervalo aberto ]a, b[. O limite de f(x) será M quando x se aproximar de a pela direita, assim:

se, para todo ε > 0, existir δ > 0, tal que se 0 < x - a < δ então |f(x) - M| < ε.

Se f é uma função definida em um intervalo aberto ]b, a[. O limite de f(x) será M quando x se aproximar de a pela esquerda, assim:

se, para todo ε > 0, existir δ > 0, tal que se -δ < x - a < 0 então |f(x) - M| < ε.

Teorema

Seja I um intervalo aberto contendo a e f uma função definida para x ∈ I - {a}.

Limites infinitos

Para valores próximos e à esquerda de 2, temos:

x 1 1,5 1,75 1,9 1,99 1,999
f(x) 2 8 32 200 20 000 2 000 000

e para valores próximos e à direita de 2, temos:

x 3 2,5 2,25 2,1 2,01 2,001
f(x) 2 8 32 200 20 000 2 000 000

 

Assim, podemos ver que os valores da função são cada vez maiores, à medida que x se aproxima de 2. Ou seja, podemos tornar f(x) tão grande quanto desejarmos.

 

 

lê-se "+∞" como "mais infinito" ou "infinito positivo".

 

Definição

 

Dado um intervalo aberto I que contém um real a, e dada uma função f definida em I - {a}. Dizemos que, quando x se aproxima de a, f(x) cresce ilimitadamente, assim:

se, para qualquer número P > 0, existir δ > 0 tal que se 0 < |x - a| < δ então f(x) > P.

 

O que ocorre com a função quando x se aproxima de a é indicado pelo símbolo "+∞", que não representa nenhum número real.

 

 

Os valores da função g são opostos aos valores da função f. Portanto, quando x se aproxima de 2, os valores de g(x) decrescem ilimitadamente.

 

 

lê-se "-∞" como "menos infinito" ou "infinito negativo".

Definição

 

Dado um intervalo aberto I que contém um real a, e dada uma função f definida em I - {a}. Dizemos que, quando x se aproxima de a, f(x) decresce ilimitadamente, assim:

se, para qualquer número P > 0, existir δ > 0 tal que se 0 < |x - a| < δ então f(x) < P.

 

O que ocorre com a função quando x se aproxima de a é indicado pelo símbolo "-∞", que não representa nenhum número real.

 

Para valores menores que 2, porém próximos a 2, temos:

x 1 1,5 1,75 1,9 1,99 1,999
f(x) -2 -4 -8 -20 -200 -2 000

e para valores maiores que 2, porém próximos a 2, temos:

x 3 2,5 2,25 2,1 2,01 2,001
f(x) 2 4 8 20 200 2 000

 

 

Se x assume valore próximos e menores que 2, os valores da função decrescem ilimitadamente e se x assume valores próximos e maiores que 2, os valores da função crescem ilimitadamente.

se, para qualquer número P > 0, existir δ > 0 tal que se 0 < x - a < δ então f(x) > P.

Limites no infinito

Para valores crescentes de x, de tal forma que x cresça ilimitadamente, temos:

x 1 5 10 100 1 000 10 000
f(x) 5 1,8 1,4 1,04 1,004 1,0004

 

 

Perceba que, ao atribuirmos a x valores positivos e crescentes, os valores da função se aproximam cada vez mais de 1, ou seja, atribuindo a x valores cada vez maiores podemos tornar f(x) tão próximo de 1 quanto quisermos.

Dado um intervalo aberto ]-∞, a[, e dada uma função f definida nesse intervalo. Dizemos que f(x) se aproxima de P quando x decresce ilimitadamente, assim:

 

se, para qualquer número ε > 0, existir N < 0 tal que se x < N então |f(x) - P| < ε.

 

 

Dada a função f(x) = x2, para todo x real. Se x assumir os valores 1, 5, 10, 100, 1 000 e sucessivamente, de forma que x cresça ilimitadamente, temos:

 

 

x 1 5 10 100 1 000
f(x) 1 25 100 10 000 1 000 000

Os valores da função crescem ilimitadamente à medida que x cresce. Podemos dizer que para tornar f(x) tão grande quanto quisermos, basta tomarmos para x valores suficientemente grandes.

 

Se atribuirmos para x valores negativos, de forma que x decresça ilimitadamente, temos:

x -1 -5 -10 -100 -1 000
f(x) 1 25 100 10 000 1 000 000

Os valores da função crescem ilimitadamente à medida que x decresce. Podemos dizer que para tornar f(x) tão grande quanto quisermos, basta tomarmos para x valores negativos cujos módulos suficientemente grandes.

Complementos

Função limitada

Dizemos que uma função f é limitada quando sua imagem é um conjunto limitado.

Se f é limitada em um conjunto A, então existem a e b reais tais que, para todo x ∈ A, vale a < f(x) < b.

Exemplo:

A função f(x) = sen x é limitada em R, pois -1 ≤ sen x ≤ 1, x ∈ R.
