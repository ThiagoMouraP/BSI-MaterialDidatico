Introdução às funções

Definição

Sendo A e B conjuntos não vazios, uma relação f de A em B recebe o nome de função definida em A com imagens em B se, e somente se, para todo x ∈ A existe um só y ∈ B tal que (x, y) ∈ f.

f é aplicação de A em B ⇔ (∀ x ∈ A, ∃ | y ∈ B | (x, y) ∈ f)

Esquema de flechas

Todo elemento de A "deve servir como ponto de partida de flecha".

Cada elemento de A "deve servir como ponto de partida de uma única flecha".

Uma relação f não é função se não satisfizer uma das condições acima, ou seja:

  1. se houver elemento de A do qual não parta nenhuma flecha ou
  2. se houver elemento de A do qual partam duas ou mais flechas.

Domínio e imagem

Domínio nada mais é que o conjunto de partida.

Contradomínio é o conjunto de chegada.

Conjunto imagem podemos dizer que é o conjunto dos elementos que recebem as setas.

Com isso o conjunto imagem acaba sendo um subconjunto do contradomínio da função.

Notação das funções

Há uma sentença aberta y = f(x) que expressa a lei mediante a qual, dado x ∈ A, determina-se y ∈ B tal que (x, y) ∈ f, assim

f é aplicação de A em B ⇔ (∀ x ∈ A, ∃ | y ∈ B | (x, y) ∈ f)

Imagem de um elemento

Quando (a, b) ∈ f, o elemento b é chamado imagem de a ou valor de f no elemento a, assim:

f(a) = b (f de a é igual a b)

Funções crescentes e decrescentes

Em linguagem não matemática, significa que a função é crescente se, ao aumentarmos o valor atribuído a x, o valor de y também aumenta.

E função decrescente, em linguagem não matemática, significa que se aumentarmos o valor atribuído a x o valor de y diminuirá.

Porém é bastante comum uma função ser crescente em certos subconjuntos de D e descrescente em outros.

A função afim é crescente quando o coeficiente angular a for positivo, e decrescente quando o coeficiente angular for negativo.

f(x) = c

Função identidade

Função identidade é uma aplicação f de R em R quando a cada elemento x ∈ R agrega o próprio x, ou seja:

f(x) = x

O gráfico é uma reta que possui as bissetrizes do 1° e 3° quadrantes.

A imagem é o conjunto Im = R.

Função linear é uma aplicação de R em R quando a cada elemento x ∈ R agrega o elemento ax ∈ R em que a ≠ 0 é um número real dado, ou seja:

f(x) = ax   (a ≠ 0)

Função afim

Função afim é uma aplicação de R em R quando a cada x ∈ R associa sempre o mesmo elemento (ax + b) ∈ R, em que a ≠ 0 e b são números reais dados.   f(x) = ax + b (a ≠ 0).

Exemplos:

  1. y = 4x + 3     em que a = 4   e b = 3
  2. y = -5x + 2   em que a = -5  e b = 2
  3. y = x - 1       em que a = 1   e b = -1

Caso b = 0, a função afim y = ax + b vira a função linear y = ax; logo, podemos concluir que a função linear é uma particular função afim.

"O gráfico cartesiano da função f(x) = ax + b (a ≠ 0) é uma reta."

Inequações

Definição

Dadas duas funções f(x) e g(x) cujos domínios estão contidos em R, denominamos inequação qualquer uma das seguintes sentenças:

  • f(x) < g(x)
  • f(x) > g(x)
  • f(x) ≤ g(x)
  • f(x) ≥ g(x)

Exemplos:

  1. 3x - 1 > x é uma inequação em que f(x) = 3x - 1 e g(x) = x.
  2. x² - 5 ≥ é uma inequação em que f(x) = x² -5 e g(x) = .

Numa inequação f(x) < g(x) a interseção dos seus domínios é chamada de domínio de validade.

Nos exemplos anteriores, temos:

  1. D = R ∩ R = R
  2. D = R ∩ R* = R*

Conjunto solução

Chamamos de conjunto solução o conjunto S de todos os números reais x tais que f(x) > g(x).

Exemplo: A inequação 3x + 2 > x + 4 tem o conjunto solução S = {x ∈ R | x > 2}, ou seja, para qualquer x ∈ S a sentença 3x + 2 > x + 4 é verdadeira.

Dizemos que a inequação f(x) > g(x) é impossível quando não existir número real x que faça com que a sentença f(x) > g(x) seja verdadeira, e representamos o conjunto solução por S = ∅.

Exemplo: A inequação x + 3 > x + 4 tem o conjunto solução S = ∅, pois não existe x ∈ R que faça com que a sentença seja verdadeira.

Resolver uma inequação significa determinar o seu conjunto solução. Se x ∈ R é solução da inequação, então x ∈ D (domínio de validade da inequação), com isso f(x) ∈ R e g(x) ∈ R. Portanto, temos:

x ∈ S   ⇒   x ∈ D

Inequação equivalente

Se o conjunto solução da primeira inequação é igual ao conjunto solução da segunda, dizemos que as inequações são equivalentes em D ⊂ R.

Exemplo:

4x + 9 > 0 e x + 3 > 0 são equivalentes em R, pois o conjuntos solução delas é S = {x ∈ R | x > -3}.
