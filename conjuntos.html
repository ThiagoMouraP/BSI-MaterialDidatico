Teoria dos Conjuntos

Conjunto - Elemento - Pertinência

Na teoria dos conjuntos existem três noções primitivas:

  • conjunto;
  • elemento;
  • pertinência entre elemento e conjunto.

A noção matemática de conjunto é similar a usada na linguagem comum: coleção, agrupamento, etc. Como exemplos temos:

  •   conjunto das vogais
  •   conjunto dos números ímpares positivos
  •   conjunto dos times de futebol do Rio de Janeiro

Chamamos de elemento um componente do conjunto. Para os exemplos acimas, temos os elementos:

  •   a, e, i, o, u
  •   1, 3, 5, 7, 9, 11, ...
  •   Vasco, Flamengo, Botafogo, Fluminense, ...

Normalmente um conjunto é representado por uma letra maiúscula, A, B, C, ..., e um elemento é representado por uma letra minúscula, a, b, c, d, ... .

Como exemplo, considere que B é um conjunto e y é um elemento. Se y pertence ao conjunto B, escrevemos:   y ∈ B.

Para indicar que y não é elemento do conjunto B, escrevemos:   y ∉ B.

 

Assim, na representação ao lado, temos:

b ∈ B, c ∈ B, d ∈ B   e   e ∉ B

A imagem ao lado, usada para representar o conjunto, é chamada de diagrama de Euler-Venn.

 

Descrição de um conjunto

Descrição pela citação dos elementos


A enumeração dos elementos de um conjunto é feita usando-se chaves.

Exemplos:

  •   conjunto das vogais: {a, e, i, o, u}
  •   conjunto dos algarismos romanos: {I, V, X, L, C, D, M}

Quando o conjunto é infinito, usa-se reticências no final.

Exemplos:

  •   conjunto dos números ímpares positivos: {1, 3, 5, 7, 9, 11, 13, ...}
  •   conjunto dos números primos positivos: {2, 3, 5, 7, 11, 13, ...}

Quando o conjunto é finito, mas com grande número de elementos, escreve-se os elementos iniciais seguidos por reticências e os últimos elementos.

Exemplos:

  •   conjunto dos números inteiros de 0 a 100: {0, 1, 2, 3, ..., 99, 100}
  •   conjunto dos divisores positivos de 80: {1, 2, 5, 10, ..., 80}

Descrição por uma propriedade

Quando queremos descrever um conjunto B por meio de uma propriedade característica T de seus elementos x, escrevemos:

B = {x | x tem a propriedade T}

e lemos: "B é o conjunto dos elementos x tal que x tem a propriedade T".

Exemplos:

  •   A = {x | x é estado da região Sudeste do Brasil} portanto:

      A = {Rio de Janeiro, São Paulo, Minas Gerais, Espírito Santo}

  •   B = {x | x é divisor inteiro de 5} portanto:

      B = {1, -1, 5, -5}

  •   C = {x | x é inteiro e 0 ≤ x ≤ 100} portanto:

      C = {0, 1, 2, 3, ..., 100}
