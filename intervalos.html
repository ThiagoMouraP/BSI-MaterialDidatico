Teoria dos Conjuntos

Intervalos

Intervalo aberto de extremos a e b é o conjunto

]a, b[ = {x ∈ R | a < x < b}

Intervalo fechado de extremos a e b é o conjunto

[a, b] = {x ∈ R | a ≤ x ≤ b}

Intervalo fechado à esquerda (ou aberto à direita) de extremos a e b é o conjunto

[a, b[ = {x ∈ R | a ≤ x < b}

Intervalo fechado à direita (ou aberto à direita) de extremos a e b é o conjunto

]a, b] = {x ∈ R | a < x ≤ b}

Os números reais a e b são denominados, respectivamente, extremo inferior e extremo superior do intervalo.

Exemplos:

  1.   ]3, 8[ = {x ∈ R | 3 < x < 8} é intervalo aberto.
  2.   [-3, 3] = {x ∈ R | -3 ≤ x ≤ 3} é intervalo fechado.
  3.   [4/9, 8[ = {x ∈ R | 4/9 ≤ x < 8} é intervalo fechado à esquerda.
  4.   ]-1/5, √3] = {x ∈ R | -1/5 < x ≤ √3} é intervalo fechado à direita.

Também há os "intervalos infinitos":

  1.   ]-∞, a[ = {x ∈ R | x < a}
  2.   ]-∞, a] = {x ∈ R | x ≤ a}
  3.   ]a, +∞[ = {x ∈ R | x > a}
  4.   [a, +∞[ = {x ∈ R | x ≥ a}
  5.   ]-∞, +∞[ = R

Exercícios

1. Descreva, conforme a notação da teoria dos conjuntos, os seguintes intervalos:

a) [-2, 4]

b) [0, 5[

c) ]-5, 7[

d) ]-∞, 9[

e) [2, +∞[

a) [-2, 4] = {x ∈ R | -2 ≤ x ≤ 4}

b) [0, 5[ = {x ∈ R | 0 ≤ x < 5}

c) ]-5, 7[ = {x ∈ R | -5 < x < 7}

d) ]-∞, 9[ = {x ∈ R | x < 9}

e) [2, +∞[ = {x ∈ R | x ≥ 2}

 

2. Descreva os seguintes conjuntos:

a) [0, 3] ∩ [2, 5]

b) [0, 4] ∩ ]3, 6[

c) ]-∞, 5] ∩ [2, +∞[

d) [2, 4] ∩ [1, 5] ∩ [-2, 7]

a) [2, 3]

b) ]3, 4]

c) [2, 5]

d) [1, 4]

 

3. Determine os seguintes conjuntos:

a) [-2, 4] ∪ [0, 5]

b) ]-4, 2] ∪ ]1, 6[

c) [-3, 3] ∪ [3, 4[

d) [-1/4, 0] ∪ [-4/3, -1/2]

a) [-2, 5]

b) ]-4, 6]

c) [-3, 4]

d) [-4/3, 0]

 

4. Sendo A = {x ∈ R | -2 < x ≤ 5} e B = {x ∈ R | 3 < x ≤ 8}, calcule A ∪ B.

{x ∈ R | -2 < x ≤ 8}

 

5. Estabeleça o domínio e a imagem das seguintes relações:

a) {(2, 2), (3, 5), (2, 4)}

b) {(-3, 4), (-2, 3), (5, -8), (2, 1)}

c) {(4, 3), (2, -5), (6, √2)}

d) {(2 + √3, √3), (2 - √4, 2)}

a) D = {2, 3} e Im = {2, 5, 4}

b) D = {-3, -2, 5, 2} e Im = {-8, 4, 3, 1}

c) D = {4, 2, 6} e Im = {3, -5, √2}

d) D = {2 + √3, 2 - √4} e Im = {√3, 2}
