Teoria dos Conjuntos

Conjuntos Numéricos

Conjunto dos números naturais

Conjunto dos números naturais (Ν) é o conjunto formado pelos números 0, 1, 2, 3, 4, ... .

Ν = {0, 1, 2, 3, 4, ...}

São definidas duas operações fundamentais nesse conjunto, a adição e a multiplicação, que apresentam as seguintes propriedades:

  • associativa da adição
    (a + b) + c = a + (b + c)
    para todos a, b, c ∈ Ν.

    •  

  • comutativa da adição
    a + b = b + a
    para todos a, b ∈ Ν.

    •  

  • elemento neutro da adição
    a + 0 = a
    para todo a ∈ Ν.

    •  

  • associativa da multiplicação
    (ab)c = a(bc)
    para todos a, b, c ∈ Ν.

    •  

  • comutativa da multiplicação
    ab = ba
    para todos a, b ∈ Ν.

    •  

  • elemento neutro da multiplicação
    a . 1 = a
    para todo a ∈ Ν.

    •  

  • distributiva da multiplicação relativamente à adição
    a(b + c) = ab + ac
    para todos a, b, c ∈ Ν.

Todos os próximos conjuntos a serem apresentados contêm Ν.

Conjunto dos números inteiros

Conjunto dos números inteiros (Z) é o conjunto formado pelos números ..., -4, -3, -2, -1, 0, 1, 2, 3, 4, ...

Z = {..., -4, -3, -2, -1, 0, 1, 2, 3, 4, ...}

O conjunto dos números inteiros pode ser dividido em três subconjuntos notáveis:

Z+ = {0, 1, 2, 3, 4, ...} = Ν

(conjunto dos inteiros não negativos);

Z- = {0, -1, -2, -3, -4, ...}

(conjunto dos inteiros não positivos);

Z* = {..., -4, -3, -2, -1, 1, 2, 3, 4, ...}

(conjunto dos inteiros não nulos);

Operações em Z

Além das operações de adição e multiplicação anteriormente vistas, no conjunto dos números inteiros ainda é definida a propriedade:

  • simétrico ou oposto para a adição
    Para todo a ∈ Z existe -a ∈ Z tal que
    a + (-a) = 0.

Devido a essa propriedade, podemos definir em Z a operação de subtração, estabelecendo que a - b = a + (-b) para todos a, b ∈ Z.

Conjuntos dos números racionais

Conjunto dos números racionais (Q) é o conjunto dos pares ordenados (ou frações) a/b, em que a ∈ Z e b ∈ Z*, para os quais adotam-se as seguintes definições:

  • igualdade:   a/b = c/d ⇔ ad = bc
  • adição:   a/b + c/d =   ad + bc/bd
  • multiplicação:   a/b . c/d = ac/bd

Destacamos os subconjuntos:

  • Q+ (conjunto dos racionais não negativos);
  • Q- (conjunto dos racionais não positivos);
  • Q* (conjunto dos racionais não nulos);

Em uma fração a/b, a é o numerador e b é o denominador. Se a e b são primos entre si, ou seja, se mdc(a, b) = 1, dizemos que a/b é uma fração irredutível.

Consideremos o conjunto Q' formado pelos números racionais com denominador unitário: Q' = {x/1 | x ∈ Z}. Temos:

  • a/1 = b/1 ⇔ a = b
  • a/1 + b/1 = a + b/1 ⇔ a + b = a + b
  • a/1 . b/1 = a . b/1 ⇔ a . b = a . b

Portanto, os racionais com denominador igual a 1 comportam-se para a igualdade, a adição e a multiplicação como se fossem números inteiros. Assim, fazendo o racional x/1 coincidir com o inteiro x, com isso:

Q' = Z, logo, Z ⊂ Q

Operações em Q

A adição e a multiplicação de racionais apresentam as seguintes propriedades:

  • (a/b + c/d) + e/f = a/b + (c/d + e/f)
  • a/b + c/d = c/d + a/b
  • a/b + 0 = a/b
  • a/b + (-a/b) = 0
  • (a/b . c/d) . e/f = a/b . (c/d . e/f)
  • a/b . c/d = c/d . a/b
  • a/b . 1 = a/b
  • a/b . (c/d + e/f) = a/b . c/d + a/b . e/f

em que a/b, c/d e e/f são racionais quaisquer; portanto, são válidas as mesmas propriedades vistas para os números inteiros. Além dessas, temos também a seguinte:

  • simétrico ou inverso para a multiplicação
    para todo a/b ∈ Q e a/b ≠ 0, existe
    b/a ∈ Q tal que a/b . b/a = 1.

Devido a essa propriedade, podemos definir em Q* a operação de divisão, estabelecendo que a/b : c/d = a/b . d/c para a/b e c/d racionais quaisquer não nulos.

Representação decimal

Todo número racional a/b pode ser representado por um número decimal. Para passar o número racional a/b para a forma de número decimal, basta dividir o inteiro a pelo inteiro b.

Podem ocorrer dois casos:

1°) Decimal exata - quando o número decimal tem uma quantidade finita de algarismos, diferentes de zero.

Exemplos:

1/2 = 0,5   3/1 = 3

2°) Dízima periódica - quando o número decimal tem uma quantidade infinita de algarismos que se repetem periodicamente.

Exemplos:

5/9 = 0,555... (período: 5)    61/495 = 0,1232323... (período: 23)

Todo número na forma de decimal exata ou de dízima periódica pode ser convertido à forma de fração a/b, com isso, representa um número racional.

Quando a decimal é uma dízima periódica, devemos procurar sua geratriz. Por exemplo:

1°) 0,444...

x = 0,444...
10x = 4,444...

  ⇒ 10x - x = 4 ⇒ x = 4/9

então: 0,444... = 4/9

2°) 1,78989...

x = 1,78989...
10x = 17,8989...
1000x = 1789,8989...

     
 

⇒   1000x - 10x = 1772
990x = 1772 ⇒ x = 1772/990

então: 1,78989... = 1772/990

Conjunto dos números reais

Números irracionais

São números cuja representação decimal não é periódica.

Exemplos:

  • 1,24681012
  • 4,606006000...
  • 20,6178462589211...

Conjunto dos números reais (R) é o conjunto formado por todos os números com representação decimal.

Todo número racional é número real:     Q ⊂ R

Além dos racionais, estão em R números irracionais como:

  • √2 = 1,4142136...
  • π = 3,1415926...

Se quisermos outros números irracionais, poderemos obtê-los, por exemplo, por meio da expressão √p, em que p é primo e positivo.

Além de Q, destacamos em R três subconjuntos:

  • R+ (conjunto dos reais não negativos);
  • R- (conjunto dos reais não positivos);
  • R* (conjunto dos reais não nulos).

Operações em R

As operações de adição e multiplicação em R contém as mesmas propriedades vistas para o conjunto Q. Em R também é definida a operação de subtração e em R* é definida a divisão.

 

Exercícios

1. Coloque na forma de fração irredutível os seguintes números racionais:
0,4;   0,444...;   0,323232...;   54,2;   5,423423423... .

2/5, 4/9, 32/99, 271/5, 602/111

 

2. Coloque em ordem crescente os seguintes números racionais: 15/16,   11/12,   18/19,   1,   47/48,   2/3.

2/3 < 11/12 < 15/16 < 18/19 < 47/48 < 1

 

3. Considere o número α = 1 + 4/10 + 1/102 + 1/103 + 1/104 + ... .

Se ele for racional, coloque-o na forma decimal e na forma de fração irredutível.

α é racional, α = 1,4111... = 127/90

 

4. Quais das proposições abaixo são verdadeiras?

a) 5 ∈ R

b) N ⊂ R

c) Z ⊂ R

d) 1/2 ∈ R - Q

e) √4 ∈ R - Q

f) (√2 - 3√3) ∈ R - Q

a, b, c, f
