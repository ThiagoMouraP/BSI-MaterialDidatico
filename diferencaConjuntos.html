Teoria dos Conjuntos

Diferença de conjuntos

Chama-se diferença entre A e B o conjunto formado pelos elementos de A que não pertencem a B.

A - B = {x | x ∈ A e x ∉ B}

  1.   {a, b} - {b, c, d} = {a}
  2.   {a, b} - {c, d, e} = {a, b}
  3.   {a, b, c} - {a, b, c, d, e, f} = ∅

Produto Cartesiano

Dados dois conjuntos A e B, o produto cartesiano desses dois conjuntos, denotado por AxB, será o conjunto de todos os pares ordenados (a,b), onde a ∈ A e b ∈ B.

Em símbolos, temos:

AxB= { (a,b) | a ∈ A e b ∈ B}

Exemplo:

A = {1, 2, 3} e B = {3, 4, 5}

AxB = {(1,3), (1,4), (1,5), (2,3), (2,4), (2,5), (3,3), (3,4), (3,5)}

Exercícios

1. Sejam os conjuntos A = {a, b, c, d}, B = {c, d, e, f, g} e C = {b, d, e, g}. Determine:

a) B - A

b) A - B

c) C - B

d) (A ∪ C) - B

e) A - (B ∩ C)

f) (A ∪ B) - (A ∩ C)

a) {e, f, g}

b) {a, b}

c) {b}

d) {a, b}

e) {a, b, c}

f) {a, c, e, f, g}

 

2. Classifique em V ou F as sentenças:

a) (A - B) ⊃ ∅

b) (A - B) ∪ (A ∩ B) = A

c) (A - B) ⊂ B

d) (A - B) ⊂ (A ∪ B)

admitindo que A e B são conjuntos quaisquer.

a)V    b)V    c)F    d)V

 

3. Dados os conjuntos A = {1, 2, 3, 4, 5}, B = {1, 2, 4, 6, 8} e C = {2, 4, 5, 7}, obtenha um conjunto X tal que X ⊂ A e A - X =   B ∩ C.

X = {1, 3, 5}

 

4. Sendo E = {1, 2, 3, 4, 5, 6, 7, 8},   p(y): y + 1 ≤ 6   e   F = {y ∈ E | y satisfaz p(y)}, determine ¬F.

¬F = {6, 7, 8}

 

5. Seja E = {a, {a}}. Diga quais das proposições abaixo são verdadeiras.

a) a ∈ E

b) {a} ∈ E

c) a ⊂ E

d) {a} ⊂ E

e) ∅ ∈ E

f) ∅ ⊂ E

a, b, d, f
