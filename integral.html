Integral Indefinida

Integração indefinida é a inversa da derivação, também chamada de antiderivação.

Dada uma função g(x), qualquer função f'(x) tal que f'(x) = g(x) é chamada integral indefinida ou antiderivada de f(x).

Propriedades das integrais indefinidas

Integral Definida

Seja uma função f definida em um intervalo [a, b].

Partição: uma partição de [a, b] é um conjunto P = {x0, x1, x2, ..., xi - 1, xi, ..., xn} com xi ∈ [a, b], i = 1, 2, ... n e a = x0 < x1 < x2 < ... < xi - 1 < xi < ... < xn = b.

Norma: chamamos norma da partição P o número μ, máximo do conjunto {Δ1x, Δ2x, ..., Δix, ..., Δnx} em que Δix = xi - xi - 1, i = 1, 2, ... n.
