Introdução às funções

Logaritmos

Dados dois números reais e positivos a e b, com a ≠ 1, chamamos de logaritmo de b na base a o expoente que deve ser dado à base a para que o resultado da potência seja igual a b.

Sendo a a base do logaritmo, b o logaritmando e x o logaritmo.

Exemplos:

  • 1. log2 16 = 4, pois 24 = 16

Logaritmação é a operação através da qual se determina o logaritmo de b (b ∈ R e b > 0) e o resultado é o logaritmo.

Antilogaritmo

Dados dois números reais positivos a e b com a ≠ 1; se o logaritmo de b na base a for x, então b é o antilogaritmo de x.

Exemplos:

  • 1. antilog3 3 = 27, pois log3 27 = 3

Sistemas de logaritmos

O conjunto de todos os logaritmos dos números reais positivos em uma base a (0 < a ≠ 1) é chamado de sistema de logaritmos dessa base a. Por exemplo, o conjunto formado por todos os logaritmos de base 2 dos números reais e positivos é o sistema de logaritmos na base 2.

Há dois sistemas de logaritmos que são bem importantes:

a) Sistema de logaritmos decimais - é o sistema de base 10.
O logaritmo decimal é indicado pela notação log10 x ou simplesmente log x.

b) Sistema de logaritmos neperianos - é o sistema de base e (e = 2,71828...), também chamado de sistema de logaritmos naturais.
O logaritmo neperiano é indicado pela notação loge x ou ln x.

Propriedades do logaritmos

1ª) Logaritmo do produto

"Em qualquer base a (0 < a ≠ 1), o logaritmo do produto de dois fatores reais positivos é igual à soma dos logaritmos dos fatores."

2ª) Logaritmo do quociente

"Em qualquer base a (0 a ≠ 1), o logaritmo do quociente de dois números reais positivos é igual à diferença entre o logaritmo do dividendo e o logaritmo do divisor."

3ª) Logaritmo da potência

"Em qualquer base a (0 a ≠ 1), o logaritmo de uma potência de base real e positiva e expoente real é igual ao produto do expoente pelo logaritmo da base da potência."

Função logarítmica

Chamamos função logarítmica de base a, sendo a um número real (0 < a ≠ 1), a função f de em R que associa a cada x o número loga x.

Propriedades

1ª) As funções f de em R, definida por f(x) = loga x, e g de R em , definida por g(x) = ax, são inversas uma da outra se 0 < a ≠ 1.

2ª) Se a > 1 (0 < a < 1), então a função logarítmica f(x) = loga x é crescente (decrescente).

Observações

1ª) Se a base é maior que 1, então a relação de desigualdade existente entre os logaritmos de dois números positivos tem o mesmo sentido que a relação entre esses números.

Exemplos:

  • I) 6 > 2          ⇒   log2 6 > log2 2
  • II) 0,3 < 5,6   ⇒   log5 0,3 < log5 5,6
  • III) √4 < 9     ⇒   log √4 < log 9

2ª) Se a base é menor que 1 e positiva, a relação de desigualdade entre os logaritmos de dois números positivos é de sentido contrário à que há entre esse número.

Exemplos:

3ª) Quando a base é maior que 1, os números positivos menores que 1 têm logaritmos negativos e os números maiores que 1 têm logaritmos positivos.

Se a > 1:
           0 < x < 1 ⇒ loga x < loga 1 ⇒ loga x < 0
           x > 1       ⇒ loga x > loga 1 ⇒ loga x > 0

Exemplos:

  • I) log3 0,30 < 0
  • II) log5 25 > 0
  • III) log7 √4 > 0

4ª) Quando a base for positiva e menor que 1, os números positivos menores que 1 têm logaritmos positivos e os números maiores que 1 têm logaritmos negativos.

Se 0 < a < 1:
           0 < x < 1 ⇒ loga x > loga 1 ⇒ loga x > 0
           x > 1       ⇒ loga x < loga 1 ⇒ loga x < 0

Exemplos:

  • I) log0,4 0,30 > 0
  • II) log0,2 2 < 0
  • III) log0,6 √7 < 0
