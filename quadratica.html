Introdução às funções

Função Quadrática

Função quadrática ou do 2° grau é uma aplicação f de R em R que associa a cada x ∈ R o elemento (ax² + bx + c) ∈, em que a, b e c são reais, com a ≠ 0.

f(x) = ax² + bx + c

Exemplos:

  1. f(x) = -x² + 2x - 3     em que    a = -1, b = 2, c = -3
  2. f(x) = 2x² + 3x         em que    a = 2, b = 3, c = 0
  3. f(x) = -3x² - 4          em que    a = -3, b = 0, c = -4

Concavidade

Se a > 0, a concavidade da parábola da função y = ax² + bx + c estará voltada para cima.

Se a < 0, a concavidade da parábola estará voltada para baixo.

Zeros

Os valores de x tais que f(x) = 0 são chamados de zeros ou raízes da função quadrática. São as soluções da equação ax² + bx + c = 0.

Os zeros da função do 2° grau são encontrados através da fórmula de Bhaskara ou da soma e produto. São os pontos onde o gráfico corta o eixo x.

Entendendo o gráfico da expressão ax2 + bx + c = 0

O a determina a concavidade da parábola (que é determinada por x2), ou seja, se a concavidade irá ficar para cima (a > 0) ou para baixo (a < 0).

O b determina como o gráfico irá cortar o eixo y. Se b for positivo, o gráfico irá cortar o eixo y de forma crescente. Caso b seja negativo, o gráfico irá cortar o eixo y de forma negativa. Se b = 0, então irá intersectar o eixo y de forma reta.

O c determina em que ponto o gráfico irá cortar o eixo y. Se c = 0, o gráfico irá cortar o eixo y na origem.

 

Gráfico da expressão y = x2 + 4x + 3.

No gráfico ao lado, podemos observar que as raízes são -3 e -1.

Como b é positivo, o gráfico corta o eixo y subindo, ou seja, de forma crescente.

E o gráfico corta o eixo y no ponto 3, pois c = 3.

Máximo e mínimo

O valor máximo da função y = f(x) é o yM, e só será se yM ≥ y para qualquer y ∈ Im(f). O ponto de máximo da função é o número xM que ∈ D(f) tal que yM = f(xM).

O valor mínimo da função y = f(x) é o ym, e só será se ym ≤ y para qualquer y ∈ Im(f). O ponto de mínimo da função é o número xm que ∈ D(f) tal que ym = f(xm).

Os valores de máximo e de mínimo são encontrados achando-se o vértice da parábola.

O xm e o ym são encontrados quando a > 0. E xM e yM são encontrados quando a < 0.
