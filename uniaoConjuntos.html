Teoria dos Conjuntos

União de Conjuntos

A união de B e C corresponde ao conjunto formado pelos elementos que pertencem a B ou a C.

B ∪ C = {x | x ∈ B ou x ∈ C}

B ∪ C é formado pelos elementos que pertencem a pelo menos um dos conjuntos B e C.

Portanto, x é elemento de B ∪ C   se   x ∈ B   ou   x ∈ C.

Exemplos:

  1.   {a, b, c} ∪ {c, d, e} = {a, b, c, d, e}
  2.   {a} ∪ {b, c, d} = {a, b, c, d}
  3.   ∅ ∪ {a, b, c, d, e} = {a, b, c, d, e}
  4.   ∅ ∪ ∅ = ∅

Propriedades da união

Sejam A, B e C conjuntos quaisquer. Esses conjuntos possuem as seguintes propriedades:

  1.   A ∪ A = A (idempotente)
  2.   A ∪ ∅ = A (elemento neutro)
  3.   A ∪ B = B ∪ A (comutativa)
  4.   A ∪ (B ∪ C) = (A ∪ B) ∪ C (associativa)

Interseção de conjuntos

A interseção dos conjuntos A e B corresponde ao conjunto formado pelos elementos que pertencem a A e a B.

A ∩ B = {x | x ∈ A e x ∈ B}

Os elementos que pertencem aos dois conjuntos (A e B) ao mesmo tempo, formam o conjunto A ∩ B.

Se y pertence a A e também pertence a B então y ∈ A ∩ B.

  1.   {a, b} ∩ {b, c} = {b}
  2.   {b, c, d} ∩ {c, d, e} = {c, d}
  3.   {d, e} ∩ {g, h} = ∅
  4.   {a, b, c} ∩ ∅ = ∅

Propriedades da interseção

Sejam A, B e C conjuntos quaisquer. Esses conjuntos possuem as seguintes propriedades:

  1.   A ∩ A = A (idempotente)
  2.   A ∩ U = A (elemento neutro)
  3.   A ∩ B = B ∩ A (comutativa)
  4.   (A ∩ B) ∩ C = A ∩ (B ∩ C) (associativa)

Conjuntos disjuntos

Quando os conjuntos A e B não possuem elemento comum, eles são ditos conjuntos disjuntos, ou seja, A ∩ B = ∅.

Propriedades

Sejam A, B e C conjuntos quaisquer. Esses conjuntos possuem as seguintes propriedades:

  •   A ∪ (A ∩ B) = A
  •   A ∩ (A ∪ B) = A
  •   A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
  •   A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
